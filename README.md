# MarketSpread Agent MVP  
William & Mary – Business Analytics Project

## Live Application
https://marketdemo.streamlit.app/

## Overview
The MarketSpread Agent MVP is an interactive decision-support dashboard designed to assist farmers market managers and vendors. It centralizes vendor performance, attendance tracking, and operational insights into a single, user-friendly interface.

This system demonstrates **agentic behavior** by combining structured data, conditional logic, and user-driven queries to generate real-time insights without relying on a large language model.

---

## Key Features
- Vendor performance tracking and analysis  
- Attendance monitoring and trend visibility  
- Identification of at-risk vendors  
- Upcoming market overview and operational insights  
- Interactive query interface (agent-style responses)  

---

## How to Use
Open the live application above and explore the dashboard or enter prompts such as:

- What does vendor performance look like right now?  
- Who is at risk this week?  
- What is the upcoming market weather?  
- How many vendors are expected this week?  

---

## Technical Architecture
- Built using Python and Streamlit  
- Uses CSV-based data storage for vendor, attendance, and sales data  
- Implements conditional logic (if/then) for decision routing  
- Modular structure separating vendor, weather, and attendance logic  

---

## Agentic Design
The system simulates an AI agent using deterministic logic rather than a large language model. User inputs are interpreted and routed to relevant data modules, which dynamically generate contextual responses.

This approach highlights how agentic systems can be built using structured logic while remaining transparent, efficient, and explainable.
